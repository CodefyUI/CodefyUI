# 分頁與持久化

## 多分頁工作區

CodefyUI 支援以分頁形式呈現多個獨立的畫布。每個分頁都有各自的：

- 節點、連線與排版
- 執行環境與紀錄
- 記錄的輸出與保留的權重（見 [教學檢視器](./teaching-inspector)）
- 復原/重做歷史（最多 50 步）

這讓你可以把多個實驗並排放著 — 例如一個分頁放訓練圖、另一個放推論圖 — 而它們的狀態不會互相干擾。

## 自動儲存

所有分頁都會自動存入瀏覽器的 `localStorage`，因此重新載入頁面時你的工作會被還原。這是瀏覽器本機的儲存；不會同步到伺服器。

## 匯入／匯出

你可以把任何圖匯出成 JSON 檔案，之後再匯入回來（或分享）：

- **匯出** 會把目前分頁的圖（節點、連線、參數與段落標記）寫成一個 `.json` 檔案。
- **匯入** 會把一個 `.json` 圖載入到新分頁。

後端的範例圖使用的正是同一種 JSON 格式，所以匯出的圖也可以用 **[CLI 圖形執行器](./cli-runner)** 在無介面（headless）下執行。

:::tip
因為圖就是純粹的 JSON，它們能乾淨地進行 diff 與版本控制。把圖和你的程式碼一起 commit，即可捕捉一條精確、可重現的管線。
:::
